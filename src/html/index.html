Create your program

Start by creating a program for your microcontroller. Choose the sensor type and select the options below. After a successful job submission you get a Job ID which you need to upload the program to microcontroller.

Upload the program to your microcontroller

You should get your Job ID after a successful job submission.

  1. Make sure the desired Job ID (program version) is above.
  2. Connect your microcontroller to your computer using an USB cable.
  3. After clicking the upload button select the microcontroller from the popup. Typically the correct entry says "USB Serial" or similar.
